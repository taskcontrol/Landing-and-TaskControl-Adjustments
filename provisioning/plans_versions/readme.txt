## RFG: 27-11-2019
Aqui se colocan las versiones de TaskControl (imagen .vdi), una por directorio,
con el formato: Plan,Version (ie. Starter-v19.03.001.001.001).
Tambien aqui iran los .sql correspondiente y/o scripts a correr para upgradear la imagen de version anterior a esta version.
Sirve tanto para la creacion de nuevos droplets como para los upgrades de los mismos.
Todo va en en el script upgrades_necesarios.sh que contendra los .sql y comandos necesarios para el upgrade.
